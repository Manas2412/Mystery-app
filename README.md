# Full Stack Next.js + AI Project

This repository contains a full-stack web application built with **Next.js**, featuring a custom authentication system (JWT + sessions), email verification with OTP, message APIs, database integration, and **AI-powered features**. The UI is built with modern React tools like shadcn and React Hook Form.

## Features
- Custom Auth System (Signup, Login, OTP)
- Zod schema validation
- Database connectivity
- Email & OTP verification
- AI integration
- Responsive UI with dashboard
- Real-world production patterns
